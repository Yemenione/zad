## نبذة عن المشروع

نظام "زاد" هو حل تقني ذكي مصمم لتحسين تجربة التغذية المدرسية. يهدف النظام إلى تمكين أولياء الأمور من طلب وجبات صحية لأطفالهم مع ضمان السلامة الصحية عبر نظام "حارس الصحة" (Health Guard) الذي يمنع تلقائياً طلب الوجبات التي تحتوي على مسببات حساسية للطفل.

### المميزات الرئيسية

- **نظام حارس الصحة**: فحص تلقائي لمكونات الوجبة مقابل التاريخ الصحي للطفل.
* **إدارة المقصف**: لوحة تحكم متكاملة (FilamentPHP) لمتابعة المخزون والمبيعات.
* **تطبيق الموبايل**: واجهة سهلة لأولياء الأمور للطلب والدفع الإلكتروني.
* **الدفع والمتابعة**: محاكاة بوابة دفع آمنة وتوليد رموز QR لاستلام الوجبات.
* **تنبيهات فورية**: إشعار ولي الأمر فور استلام الطفل للوجبة في المدرسة.

## طريقة التشغيل (التركيب التقني)

### المتطلبات الأساسية

- PHP 8.2+
* Composer
* Flutter SDK
* XAMPP / MySQL

### تشغيل لوحة التحكم (الباكيند)

1. انتقل إلى مجلد `backend`.
2. قم بتثبيت المكتبات: `composer install`.
3. قم بتنفيذ التهجير والبيانات التجريبية: `php artisan migrate --seed --seeder=ZaadDemoSeeder`.
4. ابدأ السيرفر: `php artisan serve`.
5. الدخول للوحة التحكم: عبر الرابط `http://127.0.0.1:8000/admin`.

### تشغيل تطبيق الموبايل (الفرونتند)

1. انتقل إلى مجلد `mobile`.
2. قم بتحميل المكتبات: `flutter pub get`.
3. ابدأ التطبيق: `flutter run`.
